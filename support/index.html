Common questions

Do I need an account to use Toolstem?

No, if you pay via x402 — just connect your MCP client to the endpoint, and your agent's wallet handles payment per call. Yes, if you use the Apify path — you need a funded Apify account.

What happens if a tool call fails?

If the server returns 5xx, no payment settles (we only settle on a 200 response). Email admin@toolstem.com if you observe a payment without data.

Can I get a higher rate limit or volume pricing?

Yes — for sustained volume above 100,000 calls/month, email admin@toolstem.com.

Where does the data come from?

Finance data comes from Financial Modeling Prep. SEC data comes directly from SEC EDGAR. Toolstem adds an MCP-compatible API surface and pay-per-call settlement on top.

Is Toolstem affiliated with Coinbase, Anthropic, or any other named company?

No. Toolstem is an independent operator. We use Coinbase CDP as our x402 facilitator and the Anthropic-led MCP standard as our protocol — the same way any developer might use AWS or Stripe.
